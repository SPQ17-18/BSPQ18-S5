Detailed Description

This window shows the form to signUp on NeverEmpty. The form has 4 fields: username, password, repeat password and email. It is the only information needed to sign up in NeverEmpty. When the register button is pressed the NeverEmptyController login function is called, the form data and the register method is passed to the controller.
