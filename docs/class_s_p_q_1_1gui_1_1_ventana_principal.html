Detailed Description

This is the main window of the application. It contains two lists: the catalog list with all the products and the shoppingcart list.

The profile button (bProfile) calls "VentanaPerfil" The sales filter button (bSales) filters the catalog list to show only products with discount. The buy button (bBuy) calls "VentanaTicket"
