Detailed Description

This window shows the last step to make a payment. VentanaPago has two modes: visa and paypal. Depending on the input attribute the form changes. On the top is always shown the shipping address that can be modified in "VentanaPerfil" The PayPal form consist of two fields: username and password. The VISA form consist of four fields: cardnumber, cardholder, expiration date and CVV. The left bottom button allows to use a new paypal user or visa data to pay. It calls "Animation" and transforms the form. The right bottom button calls the NeverEmptyController payWithPayPal(PaymentDTO) or payWithVisa(PaymentDTO) function.
