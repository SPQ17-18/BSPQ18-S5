Detailed Description

This is the main window of the application. It contains two lists: the catalog list with all the products and the shoppingcart list.

The login button (bLogin) calls the login function of the controller and sends the form data. The sign up (bSignUp) button calls "VentanaSeleccionarRegistro" window.
