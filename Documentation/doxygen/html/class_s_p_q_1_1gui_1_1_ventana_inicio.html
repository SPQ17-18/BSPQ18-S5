Detailed Description

This is the first window of the application. It contains a form with username and password fields to login. And two buttons: Login and Sign Up. The login button (bLogin) calls the login function of the controller and sends the form data. The sign up (bSignUp) button calls "VentanaSeleccionarRegistro" window.
